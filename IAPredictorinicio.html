Descripción General

Nuestro "IA Predictor" es una avanzada herramienta de predicción de precios de criptomonedas basada en inteligencia artificial. Utilizando datos históricos y algoritmos de aprendizaje automático, proporciona predicciones precisas sobre el comportamiento futuro del mercado de criptomonedas. A continuación, describimos los componentes clave y el flujo de trabajo de nuestro predictor.

Componentes Clave

Obtención de Datos:
API de CoinMarketCap (CMC): Recopilamos el precio actual de Bitcoin utilizando la API de CoinMarketCap.
Archivo de Datos Históricos: Cargamos datos históricos de precios y volúmenes desde un archivo local para un análisis exhaustivo.

Preprocesamiento de Datos

Normalización: Los precios históricos se normalizan para mejorar la capacidad de los modelos de aprendizaje automático de identificar patrones en los datos.
Creación de Características: Generamos características semanales a partir de los datos históricos para optimizar el entrenamiento de los modelos.

Modelos de Aprendizaje Automático

MLPRegressor (Perceptrón Multicapa): Utilizamos este modelo de red neuronal para realizar predicciones precisas.
RandomForestRegressor: Este modelo de bosque aleatorio complementa las predicciones del MLPRegressor para mejorar la robustez del pronóstico.
Pipeline: Implementamos un pipeline para escalar los datos y aplicar los modelos de manera eficiente.

Entrenamiento de Modelos

TimeSeriesSplit: Dividimos los datos en series temporales para asegurar un entrenamiento y validación precisos de los modelos, respetando la secuencia temporal de los datos.

Predicciones

Predicción Secuencial: Utilizamos los modelos entrenados para predecir los precios de Bitcoin para los próximos 7 días.
Consenso de Predicciones: Combinamos las predicciones de ambos modelos para obtener un pronóstico más sólido y confiable.

Estrategias de Compra/Venta

Cálculo de Precios: Determinamos los precios de compra y venta óptimos basados en las predicciones generadas por los modelos.
Estrategia Intradía: Desarrollamos estrategias basadas en pequeñas fluctuaciones intradía para maximizar las oportunidades de compra y venta, optimizando así el rendimiento de las inversiones.
