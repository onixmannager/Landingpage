Onix AiTrader

Sistema Dual de Trading Cuantitativo conectado a Binance vía WebSocket y REST API. Modelo Custodia-Cero: el bot opera sobre tus propios monederos sin retener tus activos.

Accede a la IA

Sistema Dual de Conexión

WebSocket (WSS): Canal en tiempo real para Orderbook L2 y Tickers de Binance.
REST API: Emisión y gestión de órdenes con latencia mínima.
Custodia-Cero: El bot envía firmas criptográficas; tus fondos nunca abandonan tu wallet.

Flujo de Ejecución

1. Ingesta: Captura de Orderbook L2 y Tickers en tiempo real.
2. Detección: Spread > Fee (Arbitraje) o cruce de EMA (Tendencia).
3. Risk Guard: Verificación de balance — máximo 30% comprometido.
4. Ejecución: Emisión y fill de la orden de forma automática.

Arbitraje Triangular (HFT)

Ciclo de 3 pares: Detecta ineficiencias entre tres pares relacionados (ej: EUR → BTC → SOL → EUR).
Umbral neto: Solo se activa cuando el beneficio supera el 0.10% tras comisiones.
TTL milisegundos: Las 3 órdenes se emiten secuencialmente en el menor tiempo posible. Con 1 o 2 pares configurados, este motor se deshabilita automáticamente.

Swing & Scalping

EMA(9/21): Analiza cruces en ventanas de 15 min y 4H para detectar momentum direccional.
RSI(14): Compra estructurada en sobreventa (<30) y venta programada en sobrecompra (>70).
Flexibilidad: Operable con 1, 2 o más pares de forma individualizada (ej: solo SOL/EUR).

Configuración de API Key

Lectura: ✔ Enable Reading.
Operativa: ✔ Enable Spot & Margin Trading.
Retiros: ✖ Enable Withdrawals — ESTRICTAMENTE PROHIBIDO.
Pega API Key y Secret Key en el panel de Configuración API del Dashboard.

Risk Guard Integral

Límite de exposición: El algoritmo nunca compromete más del 30% de tus activos líquidos disponibles en una sola inyección de capital.
Protección ante caídas: El sistema soporta fuertes correcciones del mercado sin riesgo de liquidación total de tu cartera.
