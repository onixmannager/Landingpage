Descripción General

Nuestro "IA Predictor" es una herramienta basada en inteligencia artificial que predice precios futuros de criptomonedas.
Utiliza datos históricos y algoritmos de aprendizaje automático para realizar estas predicciones.
Aquí te explicamos los componentes clave y el flujo de trabajo del predictor.

Componentes Clave

Obtención de Datos:
API de CoinMarketCap (CMC): Recopilamos el precio actual de Bitcoin usando la API de CoinMarketCap.
Archivo de Datos Históricos: Cargamos datos históricos de precios y volúmenes desde un archivo local.

Preprocesamiento de Datos

Normalización: Los precios históricos se normalizan para que los modelos puedan aprender mejor las relaciones en los datos.
Creación de Características: Creamos características semanales a partir de los datos históricos para entrenar los modelos.

Modelos de Aprendizaje Automático

MLPRegressor (Perceptrón Multicapa): Un modelo de red neuronal utilizado para hacer predicciones.
RandomForestRegressor: Un modelo de bosque aleatorio utilizado para hacer predicciones.
Pipeline: Utilizamos un pipeline para escalar los datos y aplicar el modelo.

Entrenamiento de Modelos

TimeSeriesSplit: Dividimos los datos en series temporales para entrenar y validar los modelos.

Predicciones

Predicción Secuencial: Utilizamos los modelos entrenados para predecir los precios de Bitcoin para los próximos 7 días.
Consenso de Predicciones: Calculamos un consenso entre las predicciones de ambos modelos para obtener un pronóstico más robusto.

Estrategias de Compra/Venta

Cálculo de Precios: Determinamos los precios de compra y venta óptimos basados en las predicciones de los modelos.
Estrategia Intradía: Calculamos estrategias basadas en pequeñas fluctuaciones intradía para maximizar las oportunidades de compra y venta.
