IA Predictor Crypto

Predicciones precisas de precios de criptomonedas basadas en inteligencia artificial. Modelos MLPRegressor y RandomForest entrenados con datos históricos.

Accede a la IA

Obtención de Datos

API de CoinMarketCap: Recopilamos el precio actual de Bitcoin.
Datos históricos: Archivo local con precios y volúmenes para análisis exhaustivo.

Preprocesamiento

Normalización: Escalamos los precios para mejorar la capacidad de los modelos.
Características semanales: Generamos features temporales para optimizar el entrenamiento.

Modelos de IA

MLPRegressor: Red neuronal multicapa para predicciones precisas.
RandomForestRegressor: Bosque aleatorio que complementa y robustece el pronóstico.
Pipeline: Escalado y aplicación eficiente de los modelos.

Entrenamiento

TimeSeriesSplit: Validación respetando la secuencia temporal para asegurar precisión.

Predicciones

Predicción secuencial: Pronóstico de precios para los próximos 7 días.
Consenso de modelos: Combinamos MLP y RandomForest para obtener un resultado más sólido.

Estrategias de Trading

Precios óptimos: Determinamos puntos de compra/venta basados en las predicciones.
Estrategia intradía: Aprovechamos pequeñas fluctuaciones para maximizar rendimiento.
